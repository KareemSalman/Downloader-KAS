# Downloader - KAS

A professional Python-based tool to download videos and playlists from YouTube and other platforms with a modern GUI.

---

## ✨ Features

- **Download Options:** Supports downloading individual videos or full playlists.
- **Quality Control:** Choose your preferred video resolution before downloading.
- **Multi-URL Support:** You can add and fetch multiple links at once (Batch Download).
- **Social Media Support:** Also supports downloading from **Facebook** and **Instagram**.
- **Audio Extraction:** Option to download only the audio in **MP3** format.
- **Modern GUI:** Built with a sleek, user-friendly graphical interface.

## ⚠️ Known Issues
- **Restart Button (EXE Mode):** Currently, the "Restart" button may encounter an error in the standalone `.exe` version due to temporary file cleanup by PyInstaller. 
- **Workaround:** If you need to restart the app, simply close it and open the `.exe` file again manually. 
- *Note: This feature works perfectly when running the source code directly via Python.*
